# 毛绒 ERP 主流程

## 当前正式主线

`客户 / 款式立项 -> 材料 BOM -> 辅材 / 包材采购 -> 加工合同 / 委外下单 -> 生产排单 / 跟进 / 延期 / 返工 / 异常 -> 仓库收发 / 成品入库 / 待出货 -> 对账 / 结算 / 应收应付`

## 真实来源

- 正式汇报版 PDF：老板流程、业务 / PMC / 生产经理桌面 + 手机端示意
- 加工合同 PDF：合同头、合同行、条款、回货日期、结算方式
- 材料 Excel：BOM 明细、采购汇总、色卡、作业指导书
- 加工 / 辅包材 Excel：委外加工、供应商资料、辅包材采购
- 生产订单截图：订单字段、数量、出货日期、未出货数

## 阶段说明

### 客户 / 款式立项

- 业务收口客户、订单、款式编号、产品编号、交期
- 老板审批包装材料打单表和生产资料

### 材料 BOM

- 主料真源来自 `材料分析明细表`
- 辅材 / 包材真源来自独立采购表
- 色卡 / 作业指导书属于附件层

### 加工合同

- 合同 PDF 是打印快照
- 加工汇总表是导入候选
- 加工商资料表是 partner 主档候选
- 首批打印模板已经落到业务页带值入口和打印中心默认样例入口

### 生产

- PMC 负责齐套和全流程跟进
- 生产经理负责机器加工 / 裁切、车缝 / 手工 / 内外发决策
- 返工和异常必须独立成日志

### 仓库 / 出货

- 仓库至少按 `主料仓`、`其他仓`、`成品仓` 三层理解
- 所有材料、半成品、成品、返工成品和外发件都必须先走仓库
- 主料仓 -> 机器加工 / 裁切 -> 车缝 -> 其他仓 -> 手工 / 组装 / 包装 -> 成品仓
- 主辅料到仓 -> IQC
- 包装材料到仓 -> 质量检查 -> 包装领用
- 成品回仓 -> 检验 -> 入成品仓 -> 出货放行

### 结算

- 合同条款明确按月对账、次月付款
- 当前先做结算入口，不做完整账务表

## 当前打印模板出口

- `采购合同`：采购 / 辅材 / 包材下单留档
- `加工合同`：委外加工下单、回签、对账条款留档
