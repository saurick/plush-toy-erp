# 移动端角色初始化

## 总原则

- 移动端与桌面端共用同一仓库和路由，不拆第二个项目。
- 手机端只做确认、提醒、进度回填和异常上报。
- 复杂批量编辑、打印、正式导入和扫码能力不在本轮范围。

## 老板 / 管理层

- 看今日延期、关键异常、应收应付摘要。
- 处理高优先级确认和异常费用审批。
- 不在手机端做复杂经营分析配置。

## 业务跟单

- 看今日待跟进款式、客户确认节点和材料缺口。
- 快速更新催样 / 催料备注。
- 不在手机端维护完整 BOM。

## 生产 / 厂长

- 更新开工、完工、延期和返工原因。
- 看班组交接和今日待办。
- 不在手机端做复杂排产编辑。

## 采购 / 辅料

- 看催料提醒、供应商回签、缺料和价格确认。
- 跟进加工合同节点。
- 不在手机端做正式 Excel 导入。

## 仓库 / 收发

- 确认收货、待入库、待备货和异常反馈。
- 查看今日待出货列表。
- 不在手机端做扫码入库。

## 财务 / 对账

- 看待确认对账、逾期应收和异常费用。
- 处理轻量审批与确认动作。
- 不在手机端做复杂利润报表。
