# 桌面单后台与移动端端口

## 桌面后台

- 继续只保留一个入口
- 不再保留角色切换、角色首页或角色入口菜单
- 桌面后台统一承载流程、真源、打印和帮助中心

## 移动端入口

| 角色 | 端口   | 负责什么                                        |
| ---- | ------ | ----------------------------------------------- |
| 老板 | `5186` | 交期风险、异常、待结算、本周重点                |
| 业务 | `5187` | 客户 / 款式 / 缺资料 / 催料 / 催合同 / 交期预警 |
| 采购 | `5188` | 缺料、到料、单价确认、回签、辅材包材确认        |
| 生产 | `5189` | 今日排产、进度回填、延期原因、返工、异常        |
| 仓库 | `5190` | 收货、备料、成品入库、待出货、异常件处理        |
| 财务 | `5191` | 待对账、待付款、异常费用、结算提醒              |
| PMC  | `5192` | 齐套推进、排产推进、延期跟进、催办、异常分发    |
| 品质 | `5193` | IQC、过程检验、返工复检、放行反馈、退回反馈     |

## 共享规则

- 八个移动端继续共享同一个后端 `8200`
- 共享同一套字段真源
- 共享同一套文档和帮助中心
- 不拆多个仓库
- 手机端运行页只保留任务、预警、通知、进度、任务处理和退出登录，不展示角色说明、端口说明、技术字段或帮助文案
- 移动端登录页与桌面后台一样提供密码登录和短信登录；密码登录使用管理员账号密码，短信登录的手机号必须先在桌面后台 `权限管理` 绑定到管理员账号
- 每个账号还必须勾选对应移动端角色权限；例如采购移动端只允许具备 `采购移动端` 授权的账号进入任务页
- 未绑定手机号、未授权当前角色、未登录或登录失效时只进入 `/admin-login`，不提前展示移动端内容；其中未绑定手机号只影响短信登录
