# 桌面单后台与移动端端口

## 桌面后台

- 继续只保留一个入口
- 不再保留角色切换、角色首页或角色入口菜单
- 桌面后台统一承载流程、真源、打印和帮助中心

## 移动端入口

| 角色 | 端口   | 负责什么                                        |
| ---- | ------ | ----------------------------------------------- |
| 老板 | `5186` | 交期风险、异常、待结算、本周重点                |
| 跟单 | `5187` | 客户 / 款式 / 缺资料 / 催料 / 催合同 / 交期预警 |
| 采购 | `5188` | 缺料、到料、单价确认、回签、辅材包材确认        |
| 生产 | `5189` | 今日排产、进度回填、延期原因、返工、异常        |
| 仓库 | `5190` | 收货、备料、成品入库、待出货、异常件处理        |
| 财务 | `5191` | 待对账、待付款、异常费用、结算提醒              |

## 共享规则

- 六个移动端继续共享同一个后端 `8200`
- 共享同一套字段真源
- 共享同一套文档和帮助中心
- 不拆多个仓库
