# 异常 / 返工 / 延期处理

> 适用对象：老板、业务、PMC、生产经理、采购、仓库、品质、财务  
> 当前用途：这一页只回答“异常从哪里起、谁处理、怎么回退、什么时候关闭”。

## 1. 异常类型总览

| 类型            | 常见触发点                                 | 主要责任角色    | 关闭条件                   |
| --------------- | ------------------------------------------ | --------------- | -------------------------- |
| 缺资料          | 立项资料不齐、包装材料前置信息缺失         | 业务 / 跟单     | 资料补齐并重新放行         |
| 缺料 / 到料异常 | 主料、辅包材、加工合同回签卡住             | 采购            | 已补料或已明确替代方案     |
| 延期            | 齐套延迟、排产延迟、回货延迟、发货延迟     | PMC + 生产经理  | 新交期已确认并回传         |
| 返工            | 品质不通过、工艺返修、数量不足             | 生产经理 + 品质 | 复检通过并回归主链         |
| 仓库异常        | 缺件、异常件、未齐套、未放行、仓间流转错误 | 仓库            | 条件补齐，可继续入库或出货 |
| 财务异常        | 未对账、异常费用、未放行                   | 财务            | 已放行或已明确退回原因     |

## 2. 处理原则

1. 异常优先作为独立事件处理，不继续塞在备注里。
2. 谁发现谁发起，但不能越过责任角色直接关闭。
3. 任何异常都要明确回退到主链哪个节点。
4. 重大异常同步老板，日常异常按责任角色闭环。

## 3. 主链上的异常回退规则

### 3.1 立项前置异常

1. 业务 / 跟单发现资料不齐时，回退到 `客户 / 款式立项`。
2. 如果涉及包装材料前置审批，必须回到老板审批节点。

### 3.2 采购 / 合同异常

1. 采购发现缺料、到料异常、回签未回时，先挂在采购链。
2. PMC 收到后决定是否暂停排产或拆批。
3. 不能直接跳过采购异常把单推给生产经理。

### 3.3 生产 / 品质异常

1. 生产经理或品质发现问题时，先挂到 `延期 / 返工 / 异常`。
2. 需要返工的任务退回生产经理。
3. 需要补料或改交期的任务同步退回 PMC。

### 3.4 仓库 / 发货异常

1. 仓库发现缺件、异常件、未齐套、仓间流转错误时，不能直接完成出货。
2. 业务确认和财务放行任一未完成，都视为发货未闭环。
3. 仓库异常要把责任回传给对应上游，而不是留在仓库单点消化。
4. 返工成品先回仓，再按返工任务重新领出，不单独绕开仓库。

### 3.5 财务 / 放行异常

1. 财务发现未对账、异常费用或条件不满足时，发出未放行反馈。
2. 未放行任务退回业务和仓库，必要时同步老板。
3. 放行前不得把异常视为“已完成”。

## 4. 典型异常链路

### 4.1 缺料链

1. PMC 或生产经理发现缺料。
2. 采购确认原因并反馈到料时间或替代方案。
3. PMC 决定是否调整排产。

### 4.2 返工链

1. 品质或生产经理发现不合格。
2. 生产经理安排返工。
3. 品质复检通过后回归主链。

### 4.3 延期链

1. PMC 汇总延期风险。
2. 生产经理或采购补充原因。
3. 业务回传客户侧结果，老板看重大风险。

### 4.4 放行失败链

1. 仓库准备出货。
2. 财务给出未放行反馈或业务未确认。
3. 任务退回业务 / 财务 / 仓库继续处理，不进入正式发货。

### 4.5 仓间流转失败链

1. 主料、裁片、半成品、返工成品或成品在主料仓、其他仓、成品仓之间流转失败。
2. 仓库先上报当前卡点，PMC 和生产经理判断是否影响生产节奏。
3. 若涉及品质问题，品质介入；若涉及缺料或数量差异，采购或生产侧补反馈。

## 5. 当前关闭条件

| 异常类型        | 关闭前必须满足什么             |
| --------------- | ------------------------------ |
| 缺资料          | 资料已补齐并完成必要审批       |
| 缺料 / 到料异常 | 到料或替代方案已确认           |
| 延期            | 新交期、责任说明、影响面已确认 |
| 返工            | 返工完成并复检通过             |
| 仓库异常        | 数量、状态、责任归属已明确     |
| 财务异常        | 已放行、已退回或已明确后续处理 |

## 6. 当前校对重点

1. 哪些异常只需要角色内闭环，哪些必须同步老板。
2. 品质是否单独作为异常反馈角色存在。
3. 发货未放行是否明确回退到业务 + 仓库 + 财务三方。
