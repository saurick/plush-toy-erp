# 系统初始化与端口说明

## 当前范围

- 桌面后台继续保持一个入口
- 桌面后台不再保留角色切换、角色首页或角色入口菜单
- 移动端按角色拆成 8 个入口和 8 个端口
- 所有入口继续共享同一个后端 `8200`
- 模板打印已落地业务页带值入口，当前只保留采购合同、加工合同两套可编辑打印模板；打印中心保留默认样例入口

## 当前已开始

- 采购合同可编辑打印工作台
- 加工合同可编辑打印工作台
- 任务 / 业务状态字典与 workflow schema 草案文档

## 当前明确不做

- 扩展硬件链路
- PDA
- 条码枪
- 图片识别
- 正式 Excel 导入落库
- PDF 坐标填充
- Excel 母版回写
- 打印结果自动留档或反向回写业务记录

## 当前数据库边界

- 默认开发数据库：`192.168.0.106:5432/plush_erp`
- 当前只有 `users / admin_users` 两张基线表
- 正式业务 schema migration 还没开始；workflow schema 先以帮助中心草案对线
