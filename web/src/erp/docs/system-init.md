# 系统初始化与多入口说明

## 当前范围

- 桌面后台继续保持一个入口
- 角色差异通过角色工作台、菜单权限、首页配置和帮助中心内容区分
- 移动端按角色拆成 6 个入口和 6 个端口
- 所有入口继续共享同一个后端 `8200`

## 端口矩阵

| 入口       | 端口   |
| ---------- | ------ |
| 桌面后台   | `5175` |
| 老板移动端 | `5186` |
| 跟单移动端 | `5187` |
| 采购移动端 | `5188` |
| 生产移动端 | `5189` |
| 仓库移动端 | `5190` |
| 财务移动端 | `5191` |

## 当前明确不做

- 拍照扫码
- PDA
- 条码枪
- 图片识别
- 正式 Excel 导入落库
- 合同打印模板与 PDF 坐标填充

## 当前数据库边界

- 默认开发数据库：`192.168.0.106:5432/plush_erp`
- 当前只有 `users / admin_users` 两张基线表
- 正式业务 schema 还没开始
