# 打印 / 合同 / 快照口径

> 适用对象：采购、财务、老板、开发、实施人员  
> 当前用途：这一页只收口正式打印快照和合同口径，不替代打印中心本身。

## 1. 当前首批正式打印对象

| 对象     | 真源文件                                                             | 当前用途                              | 关键边界                                       |
| -------- | -------------------------------------------------------------------- | ------------------------------------- | ---------------------------------------------- |
| 采购合同 | `模板-材料与加工合同.xlsx` 的 `C类辅料合同`                          | 辅材 / 包材采购下单、供应商确认、留档 | 合同头、明细、条款都按打印当下冻结             |
| 加工合同 | `模板-材料与加工合同.xlsx` 的 `B类加工合同` + `9.3加工合同-子淳.pdf` | 委外加工下单、回签、财务留档          | 合同头、合同行、条款、页底图样附件都按快照处理 |

## 2. 打印快照怎么理解

| 层级       | 典型字段                                                           | 当前规则                                       |
| ---------- | ------------------------------------------------------------------ | ---------------------------------------------- |
| 合同头快照 | 合同编号、下单日期、回货日期、对方名称、联系人、电话、地址         | 生成或打印时冻结，不被主档静默回刷             |
| 合同行快照 | 产品订单编号、产品编号、产品名称、工序、材料品名、数量、单价、金额 | 打印后按历史值保留                             |
| 条款快照   | 延期处罚、返工责任、对账周期、付款方式                             | 属于正式业务口径的一部分，不只留在帮助中心     |
| 附件快照   | 纸样 / 图样图片、备注                                              | 按打印当下保留图片与说明，不跟后续主档同步刷新 |

## 3. 采购合同当前口径

1. 采购合同服务的是辅材 / 包材采购，不是主料 BOM 真源。
2. 合同头里的供应商名称、联系人、电话、地址属于合同快照层。
3. 合同行里的 `厂商料号` 当前仍按原样本字段保留，不能默认等于正式物料主档料号。
4. 采购数量、单价、采购金额属于合同快照；即使后续采购表调整，历史合同不自动跟改。

## 4. 加工合同当前口径

1. 加工合同不是普通 Excel 台账，而是正式合同快照。
2. 合同头、合同行、结算条款都属于业务正式留档内容。
3. 加工商主档可以来自加工厂商资料，但合同联系人、地址、电话仍要保留历史快照。
4. 条款区里的延期处罚、返工责任、次月对账，是生产和财务流程的重要真源之一。

## 5. 哪些字段打印后必须冻结

1. 合同编号
2. 下单日期 / 回货日期
3. 供应商 / 加工商名称、联系人、联系电话、地址
4. 产品订单编号、产品编号、产品名称
5. 工序名称 / 材料品名 / 规格 / 单位
6. 单价、数量、金额
7. 合同条款

## 6. 哪些字段不能被主档回刷

1. 已打印合同的联系人、电话、地址
2. 已打印合同的单价、数量、金额
3. 已打印合同的条款和备注
4. 历史加工合同里的工序、回货日期和结算说明

## 7. 模板打印中心和帮助中心怎么配合

| 你要做什么                       | 去哪里                    | 说明                           |
| -------------------------------- | ------------------------- | ------------------------------ |
| 看正式版式、预览 PDF、下载或打印 | `模板打印中心`            | 这里看正式打印结果             |
| 判断字段属于主档还是快照         | 本页 + `ERP 字段联动口径` | 先分层，再决定能不能同步       |
| 判断数量 / 金额为什么不一致      | 本页 + `ERP 计算口径`     | 先确认看的是打印快照还是派生层 |

## 8. 当前明确不做

1. 不做 PDF 坐标回填。
2. 不做 Excel 母版回写。
3. 不把手机端变成打印或模板编辑端。
4. 不让历史合同跟着主档自动刷新。
