# ERP 操作教程（系统内速查）

> 适用对象：新同事、跨岗协作人员、现场带教  
> 建议阅读顺序：`ERP 流程图总览 -> ERP 操作教程 -> ERP 字段联动口径 -> ERP 计算口径`

## 1. 先认系统入口

1. 桌面后台统一从 `5175` 进入，不再保留桌面角色切换、角色首页或多后台入口。
2. 流程、字段真源、模板打印和正式文档默认都从桌面后台统一进入。
3. 六个移动端角色按端口直接访问，只承担各自角色的轻量查看、催办和回填动作。
4. 需要核模板、条款或打印快照时，直接进模板打印中心，不在帮助中心里口头推断模板逻辑。

## 2. 新同事先按这条主线理解系统

当前毛绒 ERP 的正式主链不是 `trade-erp` 的报价、外销、出运、结汇，而是：

`客户 / 款式立项 -> 材料 BOM -> 辅材 / 包材采购 -> 加工合同 / 委外下单 -> 生产排单 / 跟进 -> 仓库收发 / 成品入库 / 待出货 -> 对账 / 结算`

| 阶段            | 桌面后台先看什么                       | 当前重点动作                                                 | 当前责任角色  |
| --------------- | -------------------------------------- | ------------------------------------------------------------ | ------------- |
| 客户 / 款式立项 | `客户/供应商`、`产品`、`客户/款式立项` | 先确认客户、订单编号、产品订单编号、款式编号、产品编号和交期 | 跟单、老板    |
| 材料 BOM        | `材料 BOM`                             | 确认主料明细、损耗、组装部位、色卡和作业指导书               | 跟单、采购    |
| 辅材 / 包材采购 | `辅材/包材采购`                        | 确认包装材料、辅材规格、数量、单价、下单人                   | 采购          |
| 加工合同 / 委外 | `加工合同/委外下单`                    | 确认加工方、工序、数量、单价、回货日期和结算条款             | 采购、财务    |
| 生产            | `生产排单/跟进`                        | 看齐套、排产、延期、返工、异常                               | PMC、生产经理 |
| 仓库            | `仓库收发/成品入库`                    | 看到仓、IQC、成品回仓、待出货                                | 仓库、品质    |
| 对账 / 结算     | `对账/结算`                            | 看加工费、辅包材费用、待付款和异常费用                       | 财务          |

## 3. 桌面后台建议怎么用

| 场景                           | 优先入口           | 为什么                                                         |
| ------------------------------ | ------------------ | -------------------------------------------------------------- |
| 想先搞清主流程                 | `ERP 流程图总览`   | 先看各阶段、资料来源和 deferred 边界，不会一上来就陷到字段细节 |
| 想知道页面该怎么理解           | `ERP 操作教程`     | 这是帮助中心里的上手页，按“入口 -> 角色 -> 阶段”组织           |
| 想核编号、快照和字段真源       | `ERP 字段联动口径` | 统一回答“改上游会不会影响下游”“哪些字段不能混用”               |
| 想核数量、金额、日期和派生字段 | `ERP 计算口径`     | 统一回答 `qty / amount / date` 在不同单据里的实际语义          |
| 想核合同条款和正式模板         | `模板打印中心`     | 加工合同、采购合同等正式快照都在这里看，不在帮助中心里口头推断 |

## 5. 按阶段怎么操作

### 5.1 客户 / 款式立项

1. 先确认 `客户 / 订单编号 / 产品订单编号 / 客户订单号 / 款式编号 / 产品编号` 这些编号不是一个字段。
2. 再确认 `产品名称 / 颜色 / 出货日期 / 跟单业务人员` 是否已经齐。
3. 若资料不齐，不要直接往下补 BOM、辅材采购或加工合同；先把缺资料、催合同、包装材料打单这类前置动作挂回立项阶段。

当前要避免的误解：

- 不把 `trade-erp` 的“外销订单”概念直接搬过来替代当前立项页。
- 不把客户订单号、产品订单编号、款式编号、产品编号混成一个“产品编号”。

### 5.2 材料 BOM

1. 主料 BOM 真源来自 `材料分析明细表`，不是采购汇总表。
2. `材料分析汇总表-修改` 只作为采购派生层，不反向覆盖 BOM 明细。
3. 色卡、作业指导书、工艺说明属于资料层，不要长期沉在备注里。

当前要避免的误解：

- 不把主料、辅材、包材硬塞进同一张原始表。
- 不把 Excel 公式或手工汇总结果倒灌回 BOM 真源。

### 5.3 辅材 / 包材采购

1. 辅材 / 包材采购来自独立采购表，不属于主料 BOM 原始真源。
2. `厂商料号` 在真实样本里经常承载供应商简称，不能默认等于正式料号。
3. 下单人、联系电话、金额列都属于采购快照层，要和物料主档分开理解。

### 5.4 加工合同 / 委外下单

1. `9.3加工合同-子淳.pdf` 证明加工合同是正式单据快照，不只是 Excel 台账。
2. 加工合同头、合同行、结算条款、图样附件都按合同快照理解。
3. `加工厂商资料` 里的厂家信息是主档候选来源，但合同联系人、地址、电话仍要保留历史快照。

当前要避免的误解：

- 不把加工合同等同于一个“采购单”。
- 不把加工方主档字段和合同打印快照字段混成同一层。

### 5.5 生产排单 / 跟进

1. 当前生产侧先收口 `齐套 -> 排产 -> 延期 -> 返工 -> 异常` 这条主线。
2. 生产经理决定车缝 / 手工 / 内外发走向，PMC 负责全流程跟单和交期跟进。
3. 返工、延期、异常优先沉淀成独立日志，不继续塞在备注里。

当前要避免的误解：

- 不把扩展硬件链路、条码枪、PDA 假装成已上线能力。
- 不用现场口头约定替代正式异常记录。

### 5.6 仓库收发 / 成品入库 / 待出货

1. 主辅料到仓后先经过 `仓库 + 品质 IQC`。
2. 包装材料有独立支线，不要和主料到仓混成一个节点。
3. 发货前需要业务确认、仓库出货单和财务放行三个口径一起收口。

### 5.7 对账 / 结算

1. 当前财务页先盯加工费、辅包材费用、待付款和异常费用。
2. 结算条款真源优先看合同 PDF，不在帮助中心里口头重述后替代合同原文。
3. 正式结算单样本没到前，不急着把账务模型和数据库结构一次性做满。

## 6. 模板打印怎么配合帮助中心

| 场景                      | 去哪里看                              | 当前口径                                 |
| ------------------------- | ------------------------------------- | ---------------------------------------- |
| 想确认合同字段和条款      | `模板打印中心 -> 采购合同 / 加工合同` | 合同头、合同行、条款都按打印当下保存快照 |
| 想确认材料 / 加工汇总打印 | `模板打印中心`                        | 汇总表是派生展示，不反写覆盖原始真源     |
| 想确认字段真源            | `ERP 字段联动口径`                    | 先确定字段属于主档、快照还是派生         |
| 想确认公式和日期          | `ERP 计算口径`                        | 金额、数量、日期先看计算规则，再回模板   |

## 8. 当前明确不做

- 扩展硬件链路
- PDA
- 条码枪
- 图片识别
- 现场质检硬件闭环
- 正式 Excel 导入落库
- PDF 坐标填充
- Excel 母版回写

## 9. 常见问题速查

| 现象                         | 先看哪里                              | 当前处理                                         |
| ---------------------------- | ------------------------------------- | ------------------------------------------------ |
| 不确定该从哪个页面开始       | `ERP 流程图总览`                      | 先按主流程看阶段和责任角色                       |
| 不确定一个编号到底是哪层     | `ERP 字段联动口径`                    | 先区分订单编号、产品订单编号、款式编号、产品编号 |
| 合同字段和 Excel 不一致      | `模板打印中心` + `ERP 字段联动口径`   | 先分清主档字段和合同快照字段                     |
| 金额、数量或日期看起来不一致 | `ERP 计算口径`                        | 先核字段语义，再决定是不是缺值或残值             |
| 想知道某能力为什么没做       | `ERP 流程图总览` 和本页“当前明确不做” | 先看 deferred 边界，不按旧项目经验脑补           |
