# Excel / PDF 导入映射

## Excel

### `26029#夜樱烬色才料明细表2026-1-19.xlsx`

- `材料分析明细表` -> BOM 真源
- `材料分析汇总表-修改` -> 采购汇总派生
- `色卡` -> attachment / 色卡资料
- `Sheet1` -> 作业指导书 / SOP 资料

### `26204#抱抱猴子材料明细表2026-4-10.xlsx`

- 明细表比 26029 模板多出 `材料类别`、`颜色`、`损耗%`、双 `加工方式` 列
- 导入时必须兼容模板差异，不要硬写死一版表头

### `加工 成慧怡.xlsx`

- `委外加工汇总表` -> 加工合同行导入候选
- `加工厂商资料` -> partner 主档候选

### `辅材、包材 成慧怡.xlsx`

- 这是独立的辅材 / 包材采购导入源
- `厂商料号` 列当前样本经常实际承载供应商名称
- 金额列依赖公式缓存值

## PDF

### `9.3加工合同-子淳.pdf`

至少需要提取：

- 合同头：合同编号、加工方、委托单位、回货日期
- 合同行：产品订单编号、产品编号、产品名称、工序名称、工序类别、单价、数量、金额
- 条款：延期、质量、结算方式
- 附件：合同页底部图样

## 当前原则

- 先做映射设计，不抢跑落库
- 主数据字段和打印快照字段分开
- 模板不稳定时，先保留待确认项，不硬塞 schema
