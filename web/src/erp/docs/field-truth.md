# 字段真源对照

## 当前最关键的结论

当前不能再把下面几层编码混成一个字段：

- 客户 / 订单编号
- 产品订单编号 / 客户订单号
- 款式编号
- 产品编号 / SKU

## 字段对照

| 字段           | 当前真源                                   | 当前落点建议                                  |
| -------------- | ------------------------------------------ | --------------------------------------------- |
| 客户           | 生产订单总表截图                           | 订单 / 生产单快照，后续关联 partner           |
| 款式编号       | 材料分析明细表中的 `26029# / 26204#`       | style 主档 / BOM 头                           |
| 产品编号 / SKU | 合同 PDF、生产订单截图中的 `24594 / 25481` | 订单行 / 合同行快照                           |
| 产品订单编号   | 材料 Excel、加工合同、加工汇总表           | 订单头 / 订单行外部编号                       |
| 产品名称       | 材料 Excel、加工合同、生产订单截图         | style_name + name_snapshot 分层保存           |
| 数量           | 材料 Excel、合同、加工汇总、生产订单截图   | order_qty / contract_qty / planned_qty 分字段 |
| 单价           | 合同、加工汇总、辅包材采购表、生产订单截图 | 按加工 / 采购 / 订单三种语义拆开              |
| 出货日期       | 生产订单总表截图                           | 订单 / 发货快照                               |
| 回货日期       | 加工合同 PDF                               | 合同字段                                      |
| 图片 / 附件    | 合同附图、Excel 内嵌图、生产截图图片列     | attachment 表                                 |
