# ERP 计算口径

> 目的：统一毛绒 ERP 当前对 `数量 / 金额 / 日期 / 派生字段 / 打印快照` 的理解，避免把不同语义的字段硬并成一个 `qty / amount / date`。

## 1. 先看统一口径

- 数量字段必须拆场景，不保留单一裸 `qty`。
- 合同金额、合同单价、合同数量优先按合同快照理解，不因后续主档调整回写历史打印件。
- 材料 / 加工汇总表属于派生层，不反写覆盖 BOM、合同或订单真源。
- 出货日期、回货日期、下单日期、源文件日期不是同一个日期语义。
- 对于 Excel 原件里的公式列，当前优先尊重原件公式缓存值，不用手工反推去覆盖原样本。
- 通用业务记录页当前只在 `business_record_items` 行内金额为空时按 `数量 × 单价` 派生金额；如果行金额或表头金额已手工填写，按当前填写值保存，不用派生值覆盖。

## 2. 数量口径怎么拆

| 字段语义               | 当前来源                            | 当前理解                                                         |
| ---------------------- | ----------------------------------- | ---------------------------------------------------------------- |
| 订单数量               | 生产订单总表截图、材料 Excel 表头   | 属于订单 / 立项层数量，不等于加工数量、采购数量或入库数量        |
| 备品数量               | `26204` 材料分析表中的 `含备品30`   | 属于订单附加数量快照，不自动并入所有下游数量字段                 |
| 单位用量               | `材料分析明细表`                    | 属于 BOM 行级字段                                                |
| 总用量含损耗           | `材料分析明细表`                    | 属于 BOM 派生结果，按原表计算结果理解                            |
| 采购数量               | `材料分析汇总表`、辅材 / 包材采购表 | 属于采购派生或采购快照，不反写覆盖 BOM 真源                      |
| 委托加工数量           | 加工合同 PDF、加工汇总表            | 属于合同快照字段                                                 |
| 生产数量               | 生产订单总表截图                    | 属于生产执行快照，不等于订单原始数量                             |
| 未出货数               | 生产订单总表截图                    | 属于经营 / 生产报表快照，不抢跑用其他表逆推                      |
| 到仓 / 入库 / 出货数量 | 仓库与执行层结果                    | 属于执行结果，后续应单独建仓库移动或库存快照，不回写订单原始数量 |

## 3. 当前已确认的数量规则

| 口径                   | 当前规则                                             | 来源                         |
| ---------------------- | ---------------------------------------------------- | ---------------------------- |
| 主料采购汇总数量       | 由 BOM 明细派生，不是主料真源                        | 材料分析汇总表、模板打印口径 |
| 材料分析汇总表采购数量 | `总用数量 - 库存材料数量`                            | 模板打印与字段口径           |
| 辅包材采购数量         | 以采购表当前列值或公式缓存值为准                     | `辅材、包材 成慧怡.xlsx`     |
| 委托加工数量           | 以合同 / 加工汇总表当前快照为准                      | 加工合同 PDF、加工汇总表     |
| 未出货数               | 优先按经营报表当前快照理解                           | 生产订单总表截图             |
| 通用业务记录表头数量   | 表头为空时按明细行数量合计回写；表头已填时保留手工值 | 桌面业务页保存链路           |

## 4. 金额口径怎么拆

| 字段语义        | 当前来源                               | 当前理解                                                                     |
| --------------- | -------------------------------------- | ---------------------------------------------------------------------------- |
| 订单单价        | 生产订单总表截图                       | 属于订单经营快照，不等于采购单价或加工单价                                   |
| 主料 / 辅材单价 | 辅材 / 包材采购表、主料采购汇总        | 属于采购快照                                                                 |
| 加工单价        | 加工合同 PDF、加工汇总表               | 属于合同快照                                                                 |
| 委托加工金额    | 加工合同 PDF、加工汇总表               | 优先按合同或 Excel 公式缓存值理解                                            |
| 辅包材金额      | 辅材 / 包材采购表、采购合同工作台      | 优先取 Excel 公式缓存值或合同快照确认金额，不强制用数量 × 单价反推覆盖原样本 |
| 结算金额        | 合同条款、加工汇总、辅包材采购金额线索 | 当前先收口到结算入口，不抢跑建完整账务算法                                   |

## 5. 当前已确认的金额规则

| 口径                 | 当前规则                                                                       | 来源                     |
| -------------------- | ------------------------------------------------------------------------------ | ------------------------ |
| 委外加工金额         | 默认按 `单价 × 委托加工数量` 带值；若合同快照已确认金额则按当前金额列保存      | 加工合同 PDF、加工汇总表 |
| 辅包材金额           | 优先取原件公式缓存值；工作台手工改写后按当前合同快照保存                       | `辅材、包材 成慧怡.xlsx` |
| 订单单价             | 属于订单经营快照，不等于采购 / 加工单价                                        | 生产订单总表截图         |
| 历史合同金额         | 后续主档、价格或物料调整后不回写历史打印件                                     | 模板打印口径             |
| 通用业务记录明细金额 | 行金额为空且行内已有数量 / 单价时按 `数量 × 单价` 派生；行金额已填时保留手工值 | 桌面业务页保存链路       |
| 通用业务记录表头金额 | 表头为空时按明细行金额合计回写；表头已填时保留手工值                           | 桌面业务页保存链路       |

## 6. 日期口径怎么拆

| 字段            | 当前来源                    | 当前理解                                         |
| --------------- | --------------------------- | ------------------------------------------------ |
| 源文件日期      | 材料 Excel 表头、原件文件名 | 仅用于识别版本或资料时间，不直接等于业务单据日期 |
| 下单日期        | 合同或采购快照              | 属于单据生成日期快照                             |
| 回货日期        | 加工合同 PDF                | 属于加工合同承诺回货日期                         |
| 出货日期        | 生产订单总表截图            | 属于订单 / 发货快照日期                          |
| 对账 / 支付节点 | 合同条款                    | 当前先按条款快照理解，不抢跑自动化推算           |

## 7. 派生字段怎么理解

| 派生字段              | 当前口径                                   | 不该怎么理解                           |
| --------------------- | ------------------------------------------ | -------------------------------------- |
| 材料分析汇总表数量    | 从 BOM / 库存条件派生                      | 不是主料真源                           |
| 加工金额              | 从合同数量和单价派生，或沿用原件公式缓存值 | 不是主档金额                           |
| 待出货数              | 经营 / 生产报表快照                        | 不要用别的表临时逆推出一个看似正确的值 |
| 待付款 / 异常费用提醒 | 财务侧派生结果                             | 不是合同条款本身                       |

## 8. 打印和导出怎么理解

| 场景                  | 当前口径                                |
| --------------------- | --------------------------------------- |
| 采购合同打印          | 合同头、明细、条款都按当前打印快照保存  |
| 加工合同打印          | 数量、单价、金额、条款都按合同快照保存  |
| 材料 / 加工汇总表打印 | 属于派生输出，不回写覆盖 BOM 或合同真源 |
| 历史打印件重看        | 优先保留当时快照，不因主档变化自动重刷  |

## 9. 高频问题速查

| 问题                               | 先看哪里                 | 当前判断方式                                       |
| ---------------------------------- | ------------------------ | -------------------------------------------------- |
| 为什么同一个“数量”在不同页面不一样 | 本页“数量口径怎么拆”     | 先确认它是订单数量、采购数量、加工数量还是执行结果 |
| 为什么金额看起来和 Excel 不一致    | 本页“金额口径怎么拆”     | 先确认页面读的是合同快照、采购快照还是公式缓存值   |
| 为什么回货日期和出货日期不一致     | 本页“日期口径怎么拆”     | 这两个字段本来就不是同一语义                       |
| 为什么打印金额没跟主档一起更新     | 本页“打印和导出怎么理解” | 历史打印件按快照保存，不静默改写                   |
| 想核字段到底是谁带出来的           | `ERP 字段联动口径`       | 先分清真源、快照和派生                             |
