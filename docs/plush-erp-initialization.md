# 毛绒 ERP 初始化范围

## 本轮定位

本轮目标不是把 `trade-erp` 的外贸业务整套搬过来，而是先把 `plush-toy-erp` 收口成一个项目专属的 ERP 初始化框架：

- 后台主路由
- 角色工作台
- 流程总览
- 帮助中心与文档中心
- 移动端初始化页面
- 资料准备清单
- changes slug

## 已落地

- 管理员登录后默认进入 `/erp/dashboard`
- 项目内新增毛绒 ERP 的流程总览、帮助中心和文档页
- 同仓库内新增各角色的移动端工作台预览
- 已收到的合同、Excel、PDF 和截图已先挂进“资料准备”页
- 复杂任务已回写到 `docs/changes/plush-erp-bootstrap-init.md`

## 本轮不做

- 拍照扫码、PDA、条码枪、图片识别
- 正式 Excel 导入、合同打印模板和 PDF 定位填充
- 细颗粒权限矩阵、审批流和利润口径

## 端口口径

- 前端开发端口：`5175`
- 后端 HTTP 端口：`8200`
- 后端 gRPC 端口：`9200`
- PostgreSQL 宿主机映射端口：`5435`

## 后续接入顺序

1. 补更多合同原件、材料明细和辅材包材 Excel。
2. 收口字段真源、导入链路和保存链路。
3. 最后再做打印模板、扫码或图片识别。
