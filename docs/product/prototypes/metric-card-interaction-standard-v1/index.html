筛选统计卡

用于任务看板。它是真实 button，默认有动作提示、进入箭头和 `aria-pressed` 选中态。

进入动作卡 / 行级入口

用于关联记录入口、详情入口或明确的业务动作入口。默认露出动作信号，不把整块静默卡片偷偷变成可点。

实现规则

真实页面吸收时按语义选组件，不按视觉相似度偷懒复用。

只读 div、default 光标、摘要角标、无 hover 抬升。
动作 button / link、pointer 光标、动作提示、focus ring、进入信号。
筛选 button、`aria-pressed`、选中态、恢复态、URL / 状态同步。
