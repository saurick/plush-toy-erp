落地规则

进入真实实现前，先按这些规则评审菜单、页面和权限。

稳定对象进菜单 客户、订单、库存、出货、财务等稳定对象可成为主入口。
流程节点进页面 通知、检验、放行、调整、回货等流程节点优先放进 tab 或动作。
任务端不复制菜单 岗位任务按职责投影，不把桌面菜单树搬到移动端。
菜单不是权限 隐藏入口只影响可见性，后端 RBAC 和 usecase 仍是安全边界。
