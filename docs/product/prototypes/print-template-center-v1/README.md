# 模板打印中心原型 / Print Template Center Prototype

本目录归档模板打印中心的待实现 HTML 原型。它用于评审打印模板入口的信息层级、模板选择、纸面预览和打印窗口入口，不替代真实打印模板、业务字段真源、独立打印窗口或后端能力。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `index.html` | 模板打印中心待实现样板。保留左侧模板导航、右侧纸面预览、顶部打印当前模板动作和轻量状态提示；模板字段编辑和明细确认回到独立打印窗口。 |

## 待实现吸收状态

截至 2026-06-11，`index.html` 按 `待实现 / To Implement` 登记，归属 `Core`。本原型补齐打印类页面此前没有独立完整样板的问题；未获用户明确确认前，不得晋级 `当前实现 / Current`。

- 已吸收方向：模板打印中心应保持轻量，不再放静态字段映射栏，不把字段状态展示写成可配置能力。
- 已收敛模板：运行时当前已扩展为采购合同、加工合同、物料分析明细表、色卡和作业指导书；本原型仍只作为打印中心两栏骨架参考，不替代当前正式模板目录。
- 待人工确认范围：真实运行时是否按本原型进一步调整顶部动作、模板导航密度、纸面预览比例和移动端折叠方式。
- 明确不直接吸收：固定样例数据、预览里的纸面字段、窗口弹层 mock、未来模板占位和静态状态标签都不进入真实字段真源、后端 API、RBAC、菜单或打印事实。

## 当前设计结论

- 采用“模板导航 + 纸面预览”的两栏骨架；桌面优先让用户先选模板，再检查纸面布局。
- 顶部只保留当前模板打印入口和少量来源状态，避免把打印中心做成第二个字段编辑页。
- 字段编辑、明细确认、打印浏览器能力和业务页带值仍由独立打印窗口承接。
- 样品确认单暂不作为运行时候选入口、正式模板目录、模板引擎或后端事实。
- 打印预览固定浅色纸面，不跟随运行时暗色主题；页面外壳可跟随主题。
- 原型里的采购合同、加工合同样例只用于表达布局和状态，不代表客户专属合同格式或字段全集。

## 实现边界

落地时应改真实 `PrintCenterPage` 和既有打印工作台入口，不应把本目录 HTML 复制到 `web/src`。真实实现必须重新连接当前 `printTemplates` 配置、业务页带值入口、独立打印窗口、主题 token、浅色 / 暗色回归和前端测试；不得新增后端 API、RBAC、schema、migration、模板配置能力或 Fact 写入。
