# 局部动作弹窗样板 / Action Modal Standard

本目录保存后台局部动作弹窗的 `待实现 / To Implement` 标准样板。它统一列表页内的来源选择器、来源导入、列顺序、危险动作确认和状态动作说明；业务对象新建 / 编辑不再走抽屉。目录名中的 `drawer` 是历史命名，不代表业务编辑主交互。

## 文件

| 文件 | 用途 |
| --- | --- |
| `index.html` | 单文件可交互 HTML 原型，覆盖来源选择器、中型表格弹窗、列顺序调整、危险动作确认、状态动作说明、底部动作和信息边界。 |

## 标准样板口径

- 适用页面：出货、采购、质检、库存、财务等需要从列表页打开局部动作浮层的场景。
- 同类菜单默认参照本样板，不逐菜单重做原型；字段真源、动作可见性、权限、幂等、来源导入门禁、明细行校验和事实约束仍由正式 API / usecase / RBAC 决定。
- 局部动作弹窗只承接当前列表记录的辅助动作；完整新建 / 编辑必须参照 `business-form-page-standard-v1/index.html` 的同一业务表单弹窗，不另做抽屉路径。危险动作确认 / 简单提示使用小弹窗，来源选择器、列顺序和状态动作说明使用局部动作弹窗，业务单据表单使用宽业务弹窗。
- 来源选择器是第二层弹窗，只表达候选行筛选、可选状态、不可导入原因和当前动作所需列；它不编辑数量、价格、仓库、批次、收货地址等本单业务字段。
- 来源选择器必须保留分页、已选来源摘要和“清空已选”入口；已选摘要优先放在搜索框和表格之间，翻页、筛选或不可导入行置灰时，已选来源不能隐藏成用户无法撤销的残留状态。
- 来源选择器候选列应优先包含来源编号、来源行、客户 / 供应商、产品 / SKU / 材料、来源数量、已处理数量、剩余数量、状态和不可导入原因；不需要为视觉完整性展示裸数据库 ID 或无真实动作意义的字段。
- 运行态来源选择器当前优先采用来源行级候选：出货从销售订单行导入，展示订单数量、非取消出货行已生成数量和剩余可出货；采购生成入库草稿展示采购数量、已生成入库和剩余可生成。若后端只能提供单据级来源，前端不能自行拆造行级事实，只能保持单据级摘要或等待 API 补齐。
- 借鉴 `trade-erp` 时只吸收来源导入、剩余量、候选资格和已选摘要的交互模式；报关、提单、结汇、水单、磁材编码等外贸 / 行业字段不能因为来源弹窗存在就进入 plush Product Core。
- 两层为上限：主业务弹窗可以打开来源选择器；来源选择器内部若需要复杂导入、预检或明细勾选，必须在同一弹窗内分步完成，不再弹第三层。
- 导入动作只把已选来源行带回父弹窗，父弹窗继续负责本单字段维护、保存校验和残值 / 缺值防护；可导入数量、库存批次、状态边界和重复提交仍由后端守卫。
- 候选行的可选 / 不可选状态必须和后端 usecase 口径一致；前端置灰和提示只是用户反馈，不是权限、状态、幂等或数量余额的最终安全边界。
- 列顺序、状态动作说明和危险动作确认按业务列表页局部工具收口；不恢复旧业务页，也不把列偏好、删除规则或事实纠错策略写成原型真源。
- 本样板不提供通用回收站。只有对象专项已经完成后端 usecase、RBAC、审计、引用检查和测试时，才允许另行设计删除 / 恢复弹窗。
- 危险动作必须明确影响范围并二次确认；Workflow 任务关闭不等于库存、出货、财务、开票或收付款事实过账。

## 边界

- 本样板保持 `To Implement`，不晋级 `Current`。
- 不修改正式运行时代码、WorkflowUsecase、Fact usecase、后端 API、RBAC、schema 或 migration。
- 不替代 `business-form-page-standard-v1` 的业务对象新建 / 编辑弹窗骨架，也不替代字段残值 / 缺值防护规则。
- 原型中的来源选择器、明细行、列顺序、状态动作说明、危险确认和按钮状态只用于表达交互规则，真实落地仍需测试锁住后端行为。

## 验收关注点

- 局部补录、来源选择器、状态动作说明、列顺序和危险动作确认可以切换，标题、宽度、底部动作和内容密度清晰区分。
- 来源选择器能展示不可导入原因，不把不可导入行伪造成可选项，并清楚表达导入后回到父弹窗。
- 来源选择器能同时展示来源数量、已处理数量、剩余数量和本次导入后的父弹窗维护边界；缺少来源真源时保持空值或待确认，不伪造已采购、已入库、已出货、已结汇等状态。
- 分页、已选 chip 和清空入口在桌面 / 移动视口都可见，已选内容过长时不撑开弹窗。
- 危险动作必须二次确认，不可静默执行；未完成对象专项闭环时不展示删除 / 恢复入口。
- 移动端打开浮层后背景不可滚动，关闭浮层后恢复页面滚动。
- 桌面和移动视口不横向溢出，浅色 / 暗色都可读。
