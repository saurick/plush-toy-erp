## 当前硬边界

- `workflow_tasks` 是协同任务真源，不是库存、质检、出货、财务事实真源。
- ProcessRuntime 只通过明确的 domain command handler 进入业务 usecase；payload 里的 `command_key` 不被采信为事实命令。
- businessFlows / stateMachines / processPolicies 当前仍是客户配置 preview / lint / manifest 层，不是 runtime loader 或 execute 真源。
- 状态机 preview 不覆盖现有 usecase 生命周期状态规则。
- processPolicies 不允许跳过后端权限、状态、幂等、引用和 Fact 写入校验。
- 发布证据若没有 authenticated effective session active revision readback 和客户签收，只能标记 internal-only。
