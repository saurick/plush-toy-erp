# 永绅配置投影覆盖矩阵

> 本文用于标清永绅客户配置投影的 runtime-enabled 与 planned 边界，防止把规划字段当成已经由后端 effective session 执行。

## 投影层级

| 投影层 | 当前真源 | 普通账号规则 | super admin 规则 | 说明 |
| --- | --- | --- | --- | --- |
| 页面投影 | `customer_config.get_effective_session.pages` | RBAC 菜单路径 AND active revision pages | 产品核心审阅账号不被 active revision 收窄 | 页面不可见不等于后端安全边界，后端仍要 RBAC |
| 动作投影 | `customer_config.get_effective_session.actions` | 后端 RBAC AND active entitlement AND module enabled | 产品核心动作 + RBAC + active actions | 正式写按钮不能回退本地 wrapper |
| 字段投影 | `field_policies` | 仅 runtime-enabled surface 生效 | super admin 可见全部字段 | planned surface 只能作为下一步扩面清单 |
| 打印默认值 | `print_template_defaults` | 只允许买方 / 委托方默认字段 | 同样不覆盖业务快照 | 供应商 / 加工方字段必须来自业务单据快照 |

## 字段 surface

| Surface | 状态 | 字段 | 结论 |
| --- | --- | --- | --- |
| `customers.default` | runtime_enabled | customer_code / display_name | 当前后端已允许 |
| `suppliers.default` | runtime_enabled | supplier_code / supplier_type | 当前后端已允许 |
| `sales_orders.default` | runtime_enabled | order_no / source_no / expected_ship_date | 当前后端已允许 |
| `sales_order_items.default` | planned | product_no / product_name / sku_no / quantity / unit_price / amount | 下一步扩面，未当 runtime |
| `purchase_orders.default` | planned | purchase_order_no / supplier_snapshot / expected_arrival_date / lifecycle_status | 下一步扩面，未当 runtime |
| `purchase_order_items.default` | planned | product_order_no / product_no / material_name / spec / unit / quantity / unit_price / amount | 下一步扩面，未当 runtime |
| `outsourcing_orders.default` | planned | outsourcing_order_no / processor_snapshot / expected_return_date / lifecycle_status | 下一步扩面，未当 runtime |
| `quality_inspections.default` | planned | inspection_no / source_no / result / decision_note | 下一步扩面，未当 runtime |
| `inventory_lots.default` | planned | lot_no / warehouse / material / quantity / status | 下一步扩面，未当 runtime |
| `shipments.default` | planned | shipment_no / sales_order_no / warehouse / status / shipped_at | 下一步扩面，未当 runtime |
| `finance_facts.default` | planned | finance_fact_no / source_no / fact_type / amount / status | 下一步扩面，未当 runtime |

## 打印默认值边界

| 模板 | 状态 | 可默认字段 | 受保护业务快照 |
| --- | --- | --- | --- |
| `material-purchase-contract` | runtime_enabled | buyerCompany / buyerContact / buyerPhone / buyerAddress / buyerSigner | supplierName / supplierContact / supplierPhone / supplierAddress / lines |
| `processing-contract` | runtime_enabled | buyerCompany / buyerContact / buyerPhone / buyerAddress / buyerSigner | supplierName / supplierContact / supplierPhone / supplierAddress / lines |
