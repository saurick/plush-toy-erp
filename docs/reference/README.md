# 参考资料 / Reference

本目录保存外部输入、导入设计稿和其他参考材料的目录入口。

## 放什么

- 外部规划稿。
- ChatGPT / GPT 讨论导出的参考材料。
- 只作为设计输入的方案、导出资料和批次归档参考。

## 不放什么

- 当前实现真源。
- 当前 roadmap 主线。
- schema、migration、runtime、API 或 UI 执行规格。

## 是否是真源

本目录内容默认是 Reference Only。任何结论进入正式路线、架构、schema 或代码前，必须先经过当前仓库正式文档、代码和测试复核。

## 更新规则

新增、删除、重命名参考资料目录或参考文件时，必须同步检查：

- 本 README。
- `docs/document-inventory.md`。
- 若参考内容被吸收到正式路线或架构文档，还要同步对应正式文档。
