- [ ] 服务健康。
- [ ] 容器重启次数。
- [ ] 磁盘使用率和增长趋势。
- [ ] 数据库连接数。
- [ ] 慢查询摘要，如已启用。
- [ ] 最近备份状态。
- [ ] 最近恢复演练状态。
- [ ] 最近 7 天错误日志摘要。
- [ ] 登录失败次数。
- [ ] 未处理导入 unresolved queue；当前仅限 dry-run / review。
- [ ] 未处理异常任务。
- [ ] 库存负数检查。
- [ ] 预留过期检查，如当前 release 已开放预留。
- [ ] 审计日志抽查。
- [ ] 证书有效期检查。
- [ ] 巡检报告已脱敏归档。
